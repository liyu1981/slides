Performance boost

  • Xen:
    linux kernel modified, extra ring 1 level
  • OpenVZ:
    actually OS container, and rejected by Linus
  • QEMU:
    legendary Fabrice Bellard, neat code
  • KVM:
    /dev/kvm -- the door opened by kernel
  • Finally KVM + QEMU
    cpu and memory is very close to bare metal

Network freedom

  • Traditionally:
    nat, bridge, upnp
  • Morden:
    tap, virtual nic device
  • But that is not enough!
    SDN(software defined network) recently

Storage magic

  • Powerful solutions already out there years:
    NFS, iSCSI, storage over intranet.
  • Big vendors (EMC, IBM, SUN):
    volume group, software raid, snapshots.
  • Linux world acts as usual:
    Clone! LVM!

Resource Isolation
